# Calculadora de Polígonos Regulares

Este proyecto es el Entregable de la Clase 1 para la materia de Estructura de Datos.

## Características
- Menú interactivo para seleccionar Pentágono o Hexágono.
- Validación robusta de datos (no acepta números negativos ni texto).
- Estructura de datos encapsulada (`struct`).
- Cálculo automático del perímetro y área del polígono seleccionado.

## Cómo ejecutar
1. Abre la terminal en esta carpeta.
2. Ejecuta el comando: `dotnet run`
